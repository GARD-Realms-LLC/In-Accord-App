# Frequently Asked Questions (FAQ)

## General Questions

### What is the In-Accord App?

The In-Accord App is a comprehensive application suite designed to help users achieve harmony and agreement in their digital workflows. It provides tools and features to streamline collaboration and productivity.

### What license is the project under?

The In-Accord App is licensed under the GNU General Public License v3.0 (GPL-3.0). See the [LICENSE](../LICENSE) file for details.

### Is the In-Accord App free to use?

Yes, the In-Accord App is free and open-source software.

## Installation & Setup

### What are the system requirements?

- Node.js version 18 or higher
- npm (comes with Node.js) or yarn
- A modern operating system (Linux, macOS, or Windows)

### How do I install the application?

See the [Getting Started Guide](getting-started.md#installation) for detailed installation instructions.

### Can I use yarn instead of npm?

Yes, you can use either npm or yarn. The application works with both package managers.

## Development

### How do I contribute to the project?

Please read our [Contributing Guidelines](contributing.md) for information on how to contribute.

### Where can I report bugs?

Report bugs by opening an issue on our [GitHub Issues page](https://github.com/GARD-Realms-LLC/In-Accord-App/issues).

### How do I run tests?

Run the test suite with:
```bash
npm test
```

For watch mode:
```bash
npm run test:watch
```

### How do I build for production?

```bash
npm run build
```
